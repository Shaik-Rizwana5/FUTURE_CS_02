# Phishing Email Detection & Awareness Report

## Objective
Analyze a phishing email sample and identify phishing indicators.

## Tools Used
- Browser
- Sample Phishing Email
- Gamma
- GitHub

## Analysis Approach
1. Collected a phishing email sample
2. Examined the email content
3. Identified phishing indicators
4. Classified the risk level
5. Prepared prevention guidelines

## Findings
- Generic greeting ("Dear User")
- Urgent language
- Suspicious verification link
- Fear-based messaging
- Unverified sender identity

## Risk Classification
High Risk
